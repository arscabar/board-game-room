# 명화 따라그리기

## 개요

원본 그림을 보고 제한 시간 안에 따라 그린 뒤, 각 플레이어의 그림을 공개하고 유사도 퍼센트로 순위를 정하는 드로잉 파티게임입니다.

## 인원

- 1~4명

## 시간

- 고정 2분 30초
- 별도 시간 선택 없이 게임 시작과 동시에 카운트다운이 시작됩니다.

## 원본 그림

- 현재 5종이 무작위로 선택됩니다.
- 모든 플레이어는 한 판에서 같은 원본 그림을 봅니다.
- 별이 빛나는 밤
- 해바라기
- 반 고흐의 의자
- 붓꽃
- 아몬드 꽃

## 진행

1. 서버가 원본 그림 1장을 선택합니다.
2. 모든 플레이어는 같은 원본을 보며 동시에 그림을 그립니다.
3. 플레이어는 제한 시간 안에 직접 제출할 수 있습니다.
4. 시간이 끝나면 제출하지 않은 플레이어의 현재 캔버스가 자동 제출됩니다.
5. 모든 그림이 순서대로 공개되고 스캔 연출이 진행됩니다.
6. 유사도 퍼센트가 높은 순서대로 순위가 정해집니다.

## 결과 연출

- 결과 단계에서는 원본과 플레이어 그림이 갤러리 벽에 걸립니다.
- 판정 후 1등 작품만 벽에 남고 나머지 작품은 바닥으로 떨어집니다.
- 낙하 연출 후 3초 뒤 순위표와 퍼센트가 공개됩니다.
- 전체 종료 팝업은 순위표를 볼 시간이 지나고 나서 표시됩니다.

## 점수 기준

- 클라이언트가 캔버스의 평균 색상, 채도, 밝기, 채움 비율, 중심 구도, 선 사용량을 분석합니다.
- 서버는 선택된 원본 명화의 기준값과 제출 그림의 분석값을 비교합니다.
- 최종 점수는 색감 34점, 밝기 14점, 채움 20점, 구도 22점, 완성도 10점 비율로 합산됩니다.
- 결과 단계에서는 각 그림 위로 스캔 라인이 위에서 아래로 내려가며, 최종 점수까지 퍼센트가 올라갑니다.

## 승리

가장 높은 유사도 퍼센트를 받은 플레이어가 승리합니다. 최고점이 같으면 공동 승리입니다.

## 구현 메모

- 제출 전에는 다른 플레이어의 그림이 공개되지 않습니다.
- 결과 단계에서만 모든 그림 이미지가 공개됩니다.
- 채점은 서버에서 최종 계산하지만, 캔버스 분석값은 클라이언트가 전송합니다.
- 방이 게임 선택으로 돌아가거나 방이 비면 그림 데이터는 사라지는 휘발성 상태입니다.
