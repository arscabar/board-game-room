# 알까기

## 핵심 컨셉

2~4명이 원형 목재판에서 자기 왕알을 지키며 상대 왕알을 판 밖으로 밀어내는 다인전 알까기입니다.

## 구성

- 플레이어별 왕알 1개
- 플레이어별 일반알 2개
- 플레이어별 무작위 스킬 알 3개
- 특수지형: 중앙 구멍, 끈적 지형, 미끄럼 지형, 범퍼

## 진행

1. 현재 차례 플레이어가 자기 알 하나를 끌어 방향과 힘을 정합니다.
2. 서버가 충돌, 감속, 특수지형 효과를 계산합니다.
3. 왕알이 판 밖이나 중앙 구멍으로 빠진 플레이어는 탈락합니다.
4. 살아있는 다음 플레이어로 차례가 넘어갑니다.

## 왕알

- 일반 알보다 크고 무겁습니다.
- 직접 튕길 수 있지만 힘이 크게 제한되어 조금만 움직입니다.
- 왕알이 빠지면 즉시 탈락합니다.

## 스킬 알

각 플레이어는 시작 시 아래 스킬 중 3개를 무작위로 받습니다.

- 돌진: 직접 튕길 때 더 빠르게 나갑니다.
- 닻: 더 무거워 잘 밀리지 않습니다.
- 폭발: 첫 충돌 시 주변 알을 추가로 밀어냅니다.
- 방패: 한 번만 낙하를 막고 안쪽으로 되돌아옵니다.
- 탄성: 충돌 반발력이 더 큽니다.

## 승리

마지막까지 왕알이 남아 있는 플레이어가 승리합니다.
