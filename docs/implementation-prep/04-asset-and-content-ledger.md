# 에셋·콘텐츠 제작 및 권리 장부

공개 가능한 독자 버전을 만들기 위해 모든 시각·음향·퍼즐·문안의 출처를 기록한다.

## 금지 목록

- 우봉고 공식 퍼즐 마스크, 보드, 로고, 보석, 상자 문양
- 타이거 앤 드래곤 공식 동물 실루엣, 숫자 글리프, 전장 카드 이미지, 주먹 칩
- 《더 지니어스》 로고, 가넷, 방송 카드·테이블·자막·음향·영상
- 웹에서 찾은 출처 불명 이미지와 폰트

## 게임별 필요 에셋

### blind-card-duel

- 숫자 카드 앞면 1–10의 코드 기반 레이아웃
- 카드 뒷면 패턴
- 1·5·10 칩 또는 숫자 토큰
- 선 플레이어 마커
- 펠트 테이블, 팟 링, 숨김/공개 상태 아이콘
- 딜·베팅·쇼다운·폴드 효과음은 선택 사항

### parity-tile-duel

- 숫자 1–8 타일과 독자 홀·짝 특수 타일
- 공격·방어 2행 개인 보드
- 점수 토큰
- 독자 전장 카드 fixture 3종, 라이선스 결정 후 정식 데이터 교체
- 타일 충돌·배치 효과음은 직접 녹음 또는 라이선스 음원

### mosaic-rush

- 독자 폴리오미노 12종의 코드 기반 도형
- 최소 48개 독자 퍼즐과 해답 해시
- 주사위 대신 사용할 6개 문제 기호
- 9라운드 트랙과 독자 보상 토큰
- 배경 텍스처와 완료·마감 연출
