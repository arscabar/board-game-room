# 모자이크 러시

공식 제품의 명칭·퍼즐·삽화를 복제하지 않고, 동시 퍼즐 경주의 핵심 재미만 바탕으로 만든 독자 게임입니다.

## 목표

12종의 독자 조각 중 문제에 지정된 세 조각으로 가변 크기 목표판을 먼저 정확히 채웁니다. 한 매치는 9라운드이며 라운드 순위 점수는 4·3·2·1점입니다.

## 진행

1. 모든 참가자는 독립적으로 섞인 24장 퍼즐 덱에서 한 장을 받고, 공통 문양에 해당하는 윤곽을 풉니다. 144개 카드×문양 조합은 모두 서로 다른 목표 마스크와 검증된 해답을 가집니다.
2. 60초 동안 조각을 선택하고 회전·반전해 목표판에 놓습니다.
3. 모든 칸을 겹침 없이 채운 뒤 `모자이크 완성 확인`을 누릅니다. 서버가 조각 중복, 겹침, 목표 밖 배치, 누락을 검사합니다.
4. 아무도 첫 60초에 성공하지 못하면 현재 배치를 유지한 채 60초 추가 기회를 받습니다.
5. 성공 시각 순으로 점수를 얻습니다. 9라운드 뒤 최고점자가 승리합니다.
6. 최고점이 같으면 동점자만 60초 결승과 60초 추가 기회를 진행합니다. 새 퍼즐 결승을 최대 3번 반복해도 아무도 완성하지 못하면 공동 우승입니다.

다른 참가자의 배치와 퍼즐은 네트워크 공개 상태에서 제거되며, 완성 여부와 점수만 공유됩니다.
